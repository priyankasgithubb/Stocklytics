# Stocklytics

RetailPulse is a Python-based retail analytics project that analyzes fashion brand pricing and stockout trends using data visualization.

The project focuses on identifying brands with high stockout rates and understanding how pricing strategies relate to product availability.

---

## Features

- Brand Strategy Analysis
- Stockout Rate Visualization
- Price vs Stockout Scatter Plot
- Revenue-based Bubble Visualization
- Business Insight Generation

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

## Visualization

The project includes a brand strategy scatter plot where:

- X-axis represents Average Price
- Y-axis represents Stockout Rate
- Bubble size/color represents Lost Revenue
- Highlighted regions help identify high-risk brands
